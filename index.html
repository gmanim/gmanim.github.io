Graphical Manim, or Gmanim, is a frontend abstraction to Manim. It can be used to create animated math, physics, and STEM explanatory videos. No programming experience is required! You can checkout example projects over here.


Currently, Gmanim is released as a public beta, but is only supported on macOS Mojave and higher. Special thanks to 3Blue1Brown, the original creator of Manim! However, do note that 3Blue1Brown is not affiliated with and does not explicitly endorse this product.
